# well-log-petrophysical-analysis
This project performs petrophysical analysis on real well log (LAS) data using Python. It calculates shale volume, porosity, and water saturation using standard methods and identifies reservoir and net pay zones. Multi-track log plots are used for interpretation and visualization.

# Petrophysical Analysis of Reservoir Using Python

## Overview
This project performs reservoir evaluation using real well log (LAS) data.

## Features
- Volume of Shale (Vsh)
- Porosity calculation
- Water Saturation (Archie Equation)
- Net Pay identification
- Multi-track well log visualization

## Tools
- Python
- Pandas
- NumPy
- Matplotlib
- LASIO

## Output
- Well log plots
- Reservoir zones
- Net pay intervals
